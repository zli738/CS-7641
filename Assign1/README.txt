The required code and data is in https://github.com/zli738/CS-7641.

Mushroom folder has the code and data for Mushroom classification problem. Code is within Mushroom_Data.ipynb. data is mushroom_cleaned_data.csv

HeartFailure foder has code and data for Heart Failure Prediction problem. Code is within HeartFailure.ipynb. Data is heart_failure_clinical_records.csv

To run the code, user would need to install pandas, numpy, sklearn, matplotlib, and seaborn libraries. They can be installed using pip install.
