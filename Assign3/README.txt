The required code and data is in https://github.com/zli738/CS-7641.

./Assign3/mushroom folder has the code and plots for mushroom classification dataset. Code is within mushroom.ipynb. Data is mushroom_cleaned_data.csv. Plots are generated by code and are stored in /Plots folder.

./Assign3/heartFailure foder has code and plots for heart failure prediction dataset. Code is within heartFailure.ipynb. Data is heart_failure_clinical_records.csv. Plots are generated by code and are stored in /Plots folder.


To run the code, user would need to install pandas, numpy, sklearn, matplotlib, timeit and scipy libraries. They can be installed using pip install.
